libs/data-access/src/user/user-token-meta.schema.ts

authType

Authentication types for login methods. These types indicate how the user authenticated, such as email, phone, or third-party providers.
"email_password" | "phone_otp" | "email_otp" | "facebook" | "google" | "apple" | "tiktok" | "biometric"
Grant

Grant types available for tokens.

  • 'all': Grants access to all APIs
  • 'set_password': Token is valid only for setting passwords
  • 'refresh_token': Token is used to refresh access tokens
  • 'biometric_login': Token is used for biometric-based login
"all" | "set_password" | "refresh_token" | "biometric_login"
tokenType

Token types available. Used to differentiate between access, refresh, and biometric tokens.
"access_token" | "refresh_token" | "biometric_token"
